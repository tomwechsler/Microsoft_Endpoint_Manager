# Microsoft Endpoint Manager
This repo is all about Microsoft Endpoint Manager: Microsoft Configuration Manager and Microsoft Intune!

## About this repo
This repository contains a variety of resources, including scripts, tools, documentation, and best practices for managing and deploying Microsoft Endpoint Manager solutions. Whether you're an IT professional looking to streamline your device management or a developer interested in extending the capabilities of Endpoint Manager, I hope you'll find valuable content here.

## How to use this repo
Feel free to explore the different folders and files in this repository. You can clone or download the repo to your local machine to access the resources. If you find something useful, consider starring the repo to show your support!
